Episode 1: Intro to the Cyber Security Malayalam Podcast

13 June, 2020

About this Episode

This episode explains about the need a Malayalam security podcast and how this can connect deeply with the people. There are explanations and stories about hacking and cyber security in very simple language. There are numerous security professionals in Kerala and the upcoming episodes would introduce these awesome folks to the community via the podcast.
